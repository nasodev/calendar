## 1. 시작하기

### 접속 방법
웹 브라우저에서 가족달력 URL에 접속합니다.

### 시스템 요구사항
- 최신 웹 브라우저 (Chrome, Safari, Firefox, Edge)
- 인터넷 연결
- 모바일, 태블릿, PC 모두 지원

## 2. 로그인

### 로그인 방법
1. 메인 화면에서 **이름**을 입력합니다
2. **비밀번호**를 입력합니다
3. **로그인** 버튼을 클릭합니다

### 신규 사용자
- 처음 로그인하면 자동으로 가족 구성원으로 등록됩니다
- 별도의 회원가입 절차가 필요 없습니다

### 비밀번호 표시
- 비밀번호 입력칸 옆의 눈 아이콘을 클릭하면 입력한 비밀번호를 확인할 수 있습니다

## 3. 달력 보기

가족달력은 세 가지 보기 모드를 제공합니다.

### 월간 보기 (월)
- 한 달 전체 일정을 한눈에 볼 수 있습니다
- 각 날짜 칸에 일정이 표시됩니다
- 일정이 많으면 **+N** 형태로 추가 일정 수가 표시됩니다

### 주간 보기 (주)
- 일주일 일정을 시간대별로 확인할 수 있습니다
- 24시간 시간표 형태로 표시됩니다
- 각 요일의 상세 일정을 파악하기 좋습니다

### 일간 보기 (일)
- 하루 일정을 시간대별로 상세히 볼 수 있습니다
- 24시간 시간표 형태로 표시됩니다
- 시간별 일정 확인에 적합합니다

### 보기 모드 전환
- 상단의 **월 | 주 | 일** 버튼을 클릭하여 전환합니다

### 날짜 이동
| 버튼 | 기능 |
|------|------|
| **<** (이전) | 이전 기간으로 이동 (월/주/일 단위) |
| **>** (다음) | 다음 기간으로 이동 (월/주/일 단위) |
| **오늘** | 오늘 날짜로 바로 이동 |

### 날짜 표시
- **일요일**: 빨간색으로 표시
- **토요일**: 파란색으로 표시
- **오늘**: 파란색 원으로 강조 표시

## 4. 일정 관리

### 일정 추가하기

#### 방법 1: 버튼으로 추가
1. 상단의 **+ 일정 추가** 버튼을 클릭합니다
2. 일정 정보를 입력합니다
3. **저장** 버튼을 클릭합니다

#### 방법 2: 날짜 클릭으로 추가
1. 달력에서 원하는 날짜를 클릭합니다
2. 해당 날짜가 자동으로 선택된 상태로 일정 창이 열립니다
3. 일정 정보를 입력하고 저장합니다

#### 방법 3: 시간대 클릭으로 추가 (주간/일간 보기)
1. 원하는 시간대를 클릭합니다
2. 해당 날짜와 시간이 자동으로 선택됩니다
3. 일정 정보를 입력하고 저장합니다

### 일정 입력 항목

| 항목 | 설명 | 필수 여부 |
|------|------|----------|
| **제목** | 일정의 이름 | 필수 |
| **시작 날짜** | 일정 시작일 | 필수 |
| **시작 시간** | 일정 시작 시각 | 필수 (종일 아닐 때) |
| **종료 날짜** | 일정 종료일 | 필수 |
| **종료 시간** | 일정 종료 시각 | 필수 (종일 아닐 때) |
| **종일** | 하루 종일 일정 여부 | 선택 |
| **카테고리** | 일정 분류 | 선택 |
| **반복** | 반복 일정 설정 | 선택 |
| **설명** | 일정에 대한 메모 | 선택 |

### 일정 수정하기
1. 달력에서 수정할 일정을 클릭합니다
2. 일정 상세 창이 열립니다
3. 원하는 항목을 수정합니다
4. **저장** 버튼을 클릭합니다

### 일정 삭제하기
1. 달력에서 삭제할 일정을 클릭합니다
2. 일정 상세 창이 열립니다
3. **삭제** 버튼을 클릭합니다

### 일정 표시
- 각 일정은 **가족 구성원의 색상**으로 표시됩니다
- **카테고리 색상**은 일정 왼쪽에 작은 점으로 표시됩니다
- 일정을 클릭하면 상세 정보를 확인할 수 있습니다

## 5. AI 일정 등록

AI를 활용하여 자연어나 이미지로 간편하게 일정을 등록할 수 있습니다.

> ⚠️ **참고**: AI 일정 등록 기능은 로그인한 사용자만 사용할 수 있습니다. 데모 모드에서는 이 기능을 사용할 수 없습니다.

### AI 일정 등록 열기
1. 상단의 **AI 등록** 버튼을 클릭합니다 (모바일에서는 마법봉 아이콘 ✨)
2. AI 일정 등록 대화상자가 열립니다

### 텍스트로 일정 등록

자연어로 일정을 입력하면 AI가 자동으로 분석합니다.

#### 사용 방법
1. 입력창에 일정 내용을 자연어로 입력합니다
2. **분석하기** 버튼을 클릭합니다
3. AI가 분석한 결과를 확인합니다
4. 내용이 맞으면 **등록하기** 버튼을 클릭합니다

#### 입력 예시

| 입력 | 분석 결과 |
|------|----------|
| "내일 오후 3시 치과 예약" | 내일 15:00~16:00, 치과 예약 |
| "다음주 월요일 10시 팀미팅" | 다음주 월요일 10:00~11:00, 팀미팅 |
| "1월 25일 저녁 6시 가족식사" | 1월 25일 18:00~19:00, 가족식사 |
| "매주 수요일 오전 9시 영어학원" | 매주 수요일 09:00~10:00, 영어학원 (반복) |

#### 지원하는 표현
- **날짜**: 오늘, 내일, 모레, 다음주, 이번주, 특정 날짜 (1월 25일, 1/25)
- **시간**: 오전/오후 N시, N시 반, 저녁, 아침, 점심
- **반복**: 매일, 매주, 매월, 매년

### 이미지로 일정 등록

학교 알림장, 행사 포스터, 초대장 등의 이미지에서 일정을 추출할 수 있습니다.

#### 사용 방법
1. **이미지 추가** 버튼을 클릭합니다
2. 일정 정보가 포함된 이미지를 선택합니다
3. (선택) 추가 설명을 텍스트로 입력할 수 있습니다
4. **분석하기** 버튼을 클릭합니다
5. AI가 이미지에서 추출한 일정을 확인합니다
6. 내용이 맞으면 **등록하기** 버튼을 클릭합니다

#### 지원하는 이미지 유형
- 📋 학교 알림장, 가정통신문
- 🎫 행사 포스터, 초대장
- 📅 일정표, 시간표
- 📝 메모, 캡처 화면

#### 이미지 제한사항
- 최대 파일 크기: 5MB
- 지원 형식: JPG, PNG, GIF, WebP

### 일정 확인 및 수정

AI가 분석한 일정이 표시되면:

1. **일정 제목**: AI가 추출한 일정 이름
2. **날짜 및 시간**: 시작~종료 시간
3. **만료 시간**: 등록 가능 시간 (약 5분)

#### 버튼 설명
| 버튼 | 기능 |
|------|------|
| **등록하기** | 분석된 일정을 캘린더에 추가 |
| **다시 입력** | 처음으로 돌아가 다시 입력 |
| **취소** | 대화상자 닫기 |

### 오류 처리

AI 분석이 실패하면 오류 메시지가 표시됩니다.
- **다시 시도** 버튼을 클릭하여 재시도할 수 있습니다
- 입력 내용을 더 명확하게 수정해 보세요

### 팁
- 날짜와 시간을 명확하게 입력하면 더 정확한 결과를 얻을 수 있습니다
- 이미지가 흐리거나 글씨가 작으면 인식이 어려울 수 있습니다
- 여러 일정이 포함된 경우 하나씩 나누어 입력하는 것이 좋습니다

## 6. 반복 일정

정기적으로 반복되는 일정을 설정할 수 있습니다.

### 반복 유형

| 유형 | 설명 | 예시 |
|------|------|------|
| **매일** | 매일 반복 | 매일 아침 운동 |
| **매주** | 매주 특정 요일에 반복 | 매주 월요일 회의 |
| **매월** | 매월 같은 날짜에 반복 | 매월 1일 월세 납부 |
| **매년** | 매년 같은 날짜에 반복 | 생일, 기념일 |

### 매주 반복 설정
매주 반복을 선택하면 요일을 선택할 수 있습니다:
- **월 | 화 | 수 | 목 | 금 | 토 | 일**
- 여러 요일을 동시에 선택할 수 있습니다
- 예: 매주 월/수/금 학원

### 반복 종료일
- 반복 일정의 종료 날짜를 설정할 수 있습니다
- 종료일을 설정하지 않으면 무기한 반복됩니다

## 7. 카테고리 관리

일정을 카테고리별로 분류하여 관리할 수 있습니다.

### 카테고리 관리 열기
1. 상단의 **설정(⚙️)** 아이콘을 클릭합니다
2. 카테고리 관리 창이 열립니다

### 카테고리 추가
1. 카테고리 관리 창에서 새 카테고리 이름을 입력합니다
2. 원하는 색상을 선택합니다 (16가지 색상 제공)
3. 확인 버튼을 클릭합니다

### 카테고리 수정
1. 수정할 카테고리 옆의 **연필(✏️)** 아이콘을 클릭합니다
2. 이름이나 색상을 수정합니다
3. **확인(✓)** 버튼을 클릭합니다

### 카테고리 삭제
1. 삭제할 카테고리 옆의 **휴지통(🗑️)** 아이콘을 클릭합니다

### 사용 가능한 색상
빨강, 주황, 호박색, 노랑, 라임, 초록, 에메랄드, 청록, 시안, 하늘색, 파랑, 남색, 보라, 바이올렛, 자홍, 분홍

### 카테고리 활용 예시
- 🏫 학교 - 파란색
- ⚽ 운동 - 초록색
- 👨‍👩‍👧‍👦 가족 - 주황색
- 🏥 병원 - 빨간색
- 🎉 행사 - 보라색

## 8. 가족 구성원

### 구성원 식별
- 각 가족 구성원은 **고유한 색상**이 지정됩니다
- 일정에 표시되는 색상으로 누구의 일정인지 구분할 수 있습니다
- 일정 상세 보기에서 등록자 이름을 확인할 수 있습니다

### 구성원 등록
- 처음 로그인하면 자동으로 가족 구성원으로 등록됩니다
- 관리자가 미리 구성원을 등록해둘 수도 있습니다

### 색상 구분
- 각 구성원마다 다른 색상이 자동 지정됩니다
- 색상은 관리자가 변경할 수 있습니다

## 9. 데모 모드

로그인하지 않고도 가족달력의 기능을 체험할 수 있습니다.

### 데모 모드 특징
- 로그인 없이 모든 기능을 사용할 수 있습니다
- 상단에 노란색 **"Demo 모드"** 배너가 표시됩니다
- 데이터는 브라우저에 저장됩니다

### 주의사항
- 브라우저 데이터를 삭제하면 데모 모드의 일정이 사라집니다
- 다른 기기에서는 데모 모드 일정을 볼 수 없습니다
- 가족과 일정을 공유하려면 로그인이 필요합니다

### 데모 모드 종료
- 상단 배너의 **로그인** 버튼을 클릭하여 로그인할 수 있습니다

## 자주 묻는 질문 (FAQ)

### Q: 다른 기기에서도 일정을 볼 수 있나요?
**A:** 네, 로그인하면 모든 기기에서 동일한 일정을 확인할 수 있습니다.

### Q: 일정 알림 기능이 있나요?
**A:** 현재 버전에서는 알림 기능을 지원하지 않습니다.

### Q: 가족 구성원을 추가하려면 어떻게 하나요?
**A:** 새 구성원이 자신의 이름과 비밀번호로 로그인하면 자동으로 등록됩니다.

### Q: 비밀번호를 잊어버렸어요.
**A:** 관리자에게 문의하여 비밀번호를 재설정해야 합니다.

### Q: 다른 사람의 일정을 수정할 수 있나요?
**A:** 네, 가족 구성원은 모든 일정을 수정하고 삭제할 수 있습니다.

### Q: 종일 일정은 어떻게 등록하나요?
**A:** 일정 등록 시 **종일** 체크박스를 선택하면 됩니다.

### Q: 반복 일정 중 특정 날짜만 삭제할 수 있나요?
**A:** 현재는 반복 일정 전체를 삭제하거나 유지할 수만 있습니다.

### Q: AI 일정 등록이 안 보여요.
**A:** AI 일정 등록은 로그인한 사용자만 사용할 수 있습니다. 데모 모드에서는 이 기능이 표시되지 않습니다.

### Q: AI가 일정을 잘못 분석했어요.
**A:** "다시 입력" 버튼을 클릭하고 날짜와 시간을 더 명확하게 입력해 보세요. 예: "1월 25일 오후 3시 치과"

- 모든 기능이 완전히 표시됩니다
